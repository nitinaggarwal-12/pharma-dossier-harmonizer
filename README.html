🗂️ Pharma Gemini Enterprise Agent & Local Sandbox

This project implements a multi-agent system for pharmaceutical regulatory dossier analysis, designed to be deployed on Vertex AI Agent Engine, registered in Gemini Enterprise, and run locally as an interactive reviewer dashboard.

🛠️ Technical Architecture

  • Gemini Enterprise Web App: The enterprise-user conversational chat entry point.
  • Vertex AI Agent Engine: Hosts the ADK multi-agent reasoning engine app.
  • ADK Multi-Agent Orchestrator: Coordinates specialized sub-agents (Ingestion, Retrieval, Labeling, Compliance, Delta, and Reviewers).
  • Local Next.js Reviewer Dashboard: A highly responsive, premium local dashboard for regulatory QA, delta analysis, and audits.
  • FastAPI Backend Bridge: Connects the local dashboard with local databases, Document AI APIs, BigQuery indexing, and Vertex AI embedding caches.

👥 ADK Agent Orchestration

  • PharmaOrchestratorAgent (Sequential/Coordinator Agent)
    • DossierIngestionAgent (Integrates with Google Document AI)
    • ParallelRegulatoryAnalysisStage (ParallelAgent)
      • RegulatoryRetrievalAgent (Semantic RAG guidelines search)
      • DeltaAnalysisAgent (Dossier version diff calculator)
      • LabelHarmonizerAgent (Prescribing labels safety checker)
    • ComplianceAgent (Checks folder layout structures & XML schemas)
    • EctdShellAgent (Drafts initial eCTD shell outlines)
    • FinalReviewerAgent (Compiles final validated JSON report)

🚀 Local Sandbox Startup Procedure

You can run the entire stack locally on your machine in just two steps:

1. Start the FastAPI Backend Server (Port 8002)

Executes all background PDF parsers, SQLite queries, and Vector RAG tools:

uvicorn app.main:app --host 0.0.0.0 --port 8002

2. Start the Next.js Frontend Server (Port 3000)

Launches the premium interactive reviewer dashboard:

cd frontend
npm run dev

Open http://localhost:3000/ in your browser to explore!

  • Run Backup: Run python3 scratch/upload_codebase_to_drive.py inside your terminal to re-compile the zip at any time!
